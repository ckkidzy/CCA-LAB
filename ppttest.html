Microsoft PowerPoint Practical Test (20 Marks)

Instructions:

  1. Create a new PowerPoint presentation.
  2. Save the presentation as "YourName_PowerPointTest.pptx".
  3. Complete all tasks.

Tasks:

Slide Creation and Layout (5 Marks):

  • Create a presentation with at least 5 slides.
  • Use different slide layouts (e.g., Title Slide, Title and Content, Picture with Caption).

Text and Images (7 Marks):

  • Add text to each slide, using appropriate font sizes and styles.
  • Insert at least 3 relevant images from online sources or your files.
  • Format the images by resizing and positioning them correctly.

Transitions and Animations (5 Marks):

  • Apply different slide transitions to each slide.
  • Apply animations to text or images on at least 3 slides.

Presentation Features (3 Marks):

  • Insert a header or footer that includes the slide number.
  • Demonstrate how to start a slide show from the beginning.
