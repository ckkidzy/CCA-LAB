MS Word Practical Test (20 Marks)

Instructions:

  1. Create a new MS Word document.
  2. Save the document as "YourName_MSWordTest.docx".
  3. Complete all the tasks below.
  4. Ensure your work is neat and accurate.

Tasks:

Document Formatting (5 Marks):

  • Type the following text: "The Importance of Digital Literacy."
  • Apply the "Title" style to this text.
  • Set the page margins to "Narrow".
  • Change the page orientation to "Landscape".
  • Insert a page number at the bottom center of the page.

Paragraph Formatting (5 Marks):

  • Type the following paragraph: "Digital literacy is essential in today's world. It enables individuals to effectively use digital technologies. This includes the ability to find, evaluate, create, and communicate information. Developing these skills is crucial for personal and professional success."
  • Justify the paragraph.
  • Set the line spacing to 1.5.
  • Apply a 1st line indent of 0.5 inches.
  • Apply a border around the entire paragraph.

Tables (5 Marks):

  • Insert a table with 3 columns and 4 rows.
  • Enter the following data:
    • Column 1: "Item", "Laptop", "Printer", "Software"
    • Column 2: "Price", "$800", "$150", "$50"
    • Column 3: "Quantity", "1", "1", "2"
  • Format the first row of the table with a shaded background.
  • Use the word software to bold the word.

Insertions (5 Marks):

  • Insert a picture from the online pictures that relates to computers.
  • Resize the image to a reasonable size and place it to the right side of the document.
  • Insert a header containing your name and the current date.
