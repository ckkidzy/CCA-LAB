Glossary

Jump to:
A B C D E F H I K L M N O P Q R S T U V W X Z

A

Algorithm – A step-by-step procedure to solve a problem or perform a task in computing.

Application Software – Programs designed for end users, such as word processors and web browsers.

Authentication – The process of verifying the identity of a user or system.

AutoCorrect – A feature that automatically corrects common spelling and typing errors.

AutoSave – A feature that automatically saves documents periodically.

AutoSum – A function in Excel that quickly adds up a range of numbers.

Back to top

B

Backup – A copy of data stored separately to prevent loss in case of failure.

Binary Code – A system of representing text or instructions using only two digits: 0 and 1.

Bit – The smallest unit of data in computing, representing a binary digit (0 or 1).

Booting – The process of starting or restarting a computer.

Bookmark – A tool in Word that marks specific locations in a document for quick access.

Bullet Points – Symbols used to create lists in Word, PowerPoint, and other Office apps.

Back to top

C

Cache Memory – A small-sized, high-speed memory that stores frequently accessed data.

Central Processing Unit (CPU) – The brain of a computer that processes instructions.

Cloud Computing – The delivery of computing services over the internet.

Compiler – A program that translates source code into machine code.

Cybersecurity – Practices and technologies designed to protect computers and networks from attacks.

Cell – The intersection of a row and a column in an Excel worksheet.

Clipboard – A temporary storage area for copied or cut text, images, or data.

Conditional Formatting – An Excel feature that changes the appearance of cells based on specific conditions.

Column – A vertical arrangement of data in Excel, labeled with letters (A, B, C…).

Comment – A note added to a document, spreadsheet, or slide for feedback or reminders.

Back to top

D

Database – An organized collection of data stored electronically.

Debugging – The process of identifying and fixing errors in a program.

Domain Name – The human-readable address of a website (e.g., example.com).

Data Validation – A feature in Excel that restricts the type of data that can be entered in a cell.

Default Settings – The pre-set options in Office applications, which users can customize.

Drop Cap – A large, decorative capital letter at the beginning of a paragraph in Word.

Back to top

E

Encryption – A method of securing data by converting it into a coded format.

Ethernet – A technology for networking computers using wired connections.

Embed – To insert an object (such as an Excel chart) into another Office document while maintaining a link.

Excel Formula – A mathematical equation used to perform calculations in a spreadsheet.

Back to top

F

Firewall – A security system that monitors and controls incoming and outgoing network traffic.

Firmware – Software embedded into hardware devices for controlling their functions.

Folder (Directory) – A virtual container for organizing files on a computer.

File Format – The type of file used to save a document, such as .docx (Word), .xlsx (Excel), and .pptx (PowerPoint).

Find and Replace – A tool that searches for specific text and replaces it with another in Office applications.

Footer – Text or images that appear at the bottom of every page in a Word document or slide.

Format Painter – A tool that copies formatting from one part of a document to another.

Back to top

H

Hardware – The physical components of a computer, such as the monitor, keyboard, and CPU.

Hypertext Markup Language (HTML) – The standard language for creating web pages.

Header – Text or images that appear at the top of every page in a Word document or slide.

Hyperlink – A clickable link that directs users to a website, email, or another document.

Back to top

I

Input Device – A device used to enter data into a computer, such as a keyboard or mouse.

Internet – A global network connecting millions of computers worldwide.

IP Address – A unique identifier assigned to devices on a network.

Indentation – The space between the text and the margin in Word.

Inline Image – An image that is placed within a paragraph of text instead of floating freely.

Insert Tab – A ribbon tab in Office apps that allows users to add tables, pictures, charts, and more.

Back to top

K

Kernel – The core part of an operating system that manages hardware and system resources.

Keyboard Shortcut – A combination of keys that perform a specific function quickly.

Back to top

L

Local Area Network (LAN) – A network that connects computers within a limited area, like an office.

Login Credentials – A combination of username and password used to access a system.

Lookup Function – An Excel formula that retrieves data from a table or range, such as VLOOKUP or HLOOKUP.

Back to top

M

Malware – Malicious software designed to harm or exploit computer systems.

Memory (RAM) – Temporary storage used by a computer to run applications.

Motherboard – The main circuit board that connects all hardware components of a computer.

Macro – A recorded sequence of commands used to automate repetitive tasks in Office applications.

Mail Merge – A Word feature that combines a document with a data source (e.g., Excel) to create personalized letters or labels.

Margin – The space between the text and the edge of the page in Word documents.

Back to top

N

Network – A group of interconnected computers and devices.

Node – A connection point in a network.

Navigation Pane – A panel in Word that helps users navigate through headings, pages, or search results.

Numbering – A formatting feature that creates ordered lists in Word or PowerPoint.

Back to top

O

Operating System (OS) – Software that manages hardware and software resources (e.g., Windows, Linux).

Open-Source Software – Software whose source code is available for modification and distribution.

OneDrive – Microsoft’s cloud storage service for saving and sharing Office documents.

Outline View – A Word and PowerPoint view that shows the document’s structure in a list format.

Back to top

P

Phishing – A cyber attack where attackers trick users into revealing sensitive information.

Printer – A device that produces a physical copy of digital documents.

Programming Language – A language used to write software (e.g., Python, Java, C++).

Page Break – A marker in Word that starts a new page at a specific location.

Paste Special – A feature that allows users to paste copied content in different formats.

Pivot Table – A tool in Excel used to summarize and analyze large sets of data.

Placeholder – A box in PowerPoint slides where users can insert text, images, or videos.

PowerPoint Animation – A feature that adds movement to slide objects.

Print Preview – A feature that shows how a document will look when printed.

Back to top

Q

Quick Access Toolbar – A customizable toolbar in Office apps that provides quick access to frequently used commands.

Back to top

R

Router – A device that connects different networks and directs internet traffic.

RAM (Random Access Memory) – A type of volatile memory used for temporary data storage.

Ribbon – The toolbar at the top of Office apps that contains tabs and commands.

Review Tab – A section in Word, Excel, and PowerPoint used for proofreading, tracking changes, and adding comments.

Row – A horizontal line of cells in Excel, labeled with numbers (1, 2, 3…).

Back to top

S

Software – A collection of programs that tell a computer what to do.

Spreadsheet – A digital tool used to store, organize, and analyze data in tabular form.

SQL (Structured Query Language) – A language used for managing and querying databases.

Slide Master – A feature in PowerPoint that controls the design and layout of all slides.

SmartArt – A tool in Office applications that allows users to create diagrams and visual representations of information.

Sort and Filter – Excel functions that arrange or display specific data based on set criteria.

Spell Check – A tool that checks for spelling and grammar errors in Office documents.

Back to top

T

Trojan Horse – A type of malware disguised as legitimate software.

Troubleshooting – The process of diagnosing and resolving technical problems.

Table of Contents – A list of headings and page numbers automatically generated in Word.

Template – A pre-designed document, spreadsheet, or presentation that can be customized.

Text Wrapping – A formatting option that controls how text flows around images in Word.

Track Changes – A Word feature that records edits made to a document for review.

Back to top

U

URL (Uniform Resource Locator) – The address of a webpage.

USB (Universal Serial Bus) – A standard connection interface for peripherals like flash drives.

User Interface (UI) – The visual elements that allow users to interact with a computer or application.

Undo/Redo – Commands that allow users to reverse or reapply their last action.

Back to top

V

Virus – A type of malware that spreads by attaching itself to files or programs.

Virtual Machine (VM) – A software-based emulation of a computer system.

VBA (Visual Basic for Applications) – A programming language used to create macros in Office applications.

View Tab – A ribbon tab in Office applications that allows users to switch between different document views.

VLOOKUP – An Excel function that searches for a value in a table and returns corresponding data.

Back to top

W

Web Browser – A software application used to access the internet (e.g., Chrome, Firefox).

Wi-Fi – A wireless technology used to connect devices to a network.

Word Processor – Software used to create and edit text documents (e.g., Microsoft Word).

Watermark – A faint background image or text added to a Word document or PowerPoint slide for branding or security.

Word Count – A feature in Word that displays the total number of words in a document.

Back to top

X

XML (eXtensible Markup Language) – A language used to structure and store data.

Back to top

Z

Zip File – A compressed file format used to reduce file size and bundle multiple files together.

Zoom – A tool that adjusts the view size of a document, spreadsheet, or slide.
