Microsoft Excel Practical Test (20 Marks)

Instructions:

  1. Create a new Excel workbook.
  2. Save the workbook as "YourName_ExcelTest.xlsx".
  3. Complete all tasks.

Tasks:

Data Entry and Formatting (5 Marks):

  • Create a table with the following headers: "Product", "Quantity", "Unit Price", "Total Price".
  • Enter at least 5 rows of sample data.
  • Format the "Unit Price" and "Total Price" columns as currency.
  • Bold the header row.

Formulas and Functions (7 Marks):

  • In the "Total Price" column, use a formula to calculate the total price (Quantity * Unit Price).
  • At the bottom of the "Quantity" column, use the SUM function to calculate the total quantity.
  • At the bottom of the "Total Price" column, use the SUM function to calculate the grand total.
  • Use the average function to calculate the average unit price.

Charts (5 Marks):

  • Create a bar chart that displays the "Product" and "Total Price" data.
  • Add a chart title and axis labels.

Filtering and Sorting (3 Marks):

  • Sort the data by the "Product" column in alphabetical order.
  • Use a filter to display only products with a quantity greater than a specified number.
